# 🚲 따릉이 히트맵 프로젝트

**시간대별 따릉이 대여 가능성 예측 서비스**

## 📋 프로젝트 개요

서울시 따릉이(공공자전거) 스테이션의 **시간대별 대여 가능성**을 히트맵으로 시각화하고,
**실시간 잔여 자전거 현황**과 **최적 대여 시간대를 추천**해주는 데이터 파이프라인 프로젝트입니다.

### 🎯 제공 서비스

1. **스테이션 검색**: 원하는 따릉이 스테이션 선택
2. **시간별 히트맵**: 24시간 대여 가능성을 색상으로 시각화
3. **실시간 현황**: 현재 잔여 자전거 수 표시
4. **추천 메시지**: 최적 대여 시간대 안내
